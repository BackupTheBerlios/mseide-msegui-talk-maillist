On Monday, 17. January 2011 07.41:32 dan wrote:
> I usually use this function for get Idle Time(in Seconds) in delphi:
>
> Function SecondsIdle: DWord;
> var
>    liInfo: TLastInputInfo;
> begin
>    liInfo.cbSize := SizeOf(TLastInputInfo) ;
>    GetLastInputInfo(liInfo) ;
>    Result := (GetTickCount - liInfo.dwTime) DIV 1000;
> end;
>
> How to use in msegui?
>
On Windows you can probably do the same.
The MSEgui key and mouse event info records have a "timestamp" field (micro 
seconds). Trunk 4271 has tguiapplication.lastinputtimestamp for convenience.
