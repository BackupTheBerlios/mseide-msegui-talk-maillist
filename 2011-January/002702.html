DPI for screen and printer maybe different.

For  screen display :
Windows  use 96 PPI (DPI), so PPMM = 3.78 or PPCM = 38 (rounded as in table).
Macintosh use 72 PPI (DPI), so PPMM = 2.83 or PPCM = 28 (rounded too).
