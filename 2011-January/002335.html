Really, "fixed" is present on any LINUX system since is a obligatory font  
for XOrg just to start. But by default this font is a bitmap font (only  
225 bytes charset) & can't be used for displaying contents of multiple  
languages (utf8, WWB-browsers,..). For russian, this fonts if mapped to  
"*-*...*-koi8-r" and can't display russian WEB-pages in cp1251. And so on.
