Also an asking to include :

//----------------
program mseide;

{$ifdef mswindows}
  {$R mseide.res}
{$endif}

[..]
//---------------

to "mseide.pas", the SVN manistream - it'll be neccessary for win-23  
installer.

The "res" file is compiled by "windres -O res -i mseide.rc -o mseide.res"  
for the attached files.
You may put the attached res-file to "apps/ide" directory so that the IDE  
to compile once change is applied - it refers to IDE v 2.4 & 64x64 32bit  
icon with transparency (converted from the 64x64 "mseide" PNG).
