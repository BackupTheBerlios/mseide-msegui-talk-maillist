On Friday, 7. January 2011 16.36:55 dan wrote:
> Is it possible to set separate directory for backup files (.bak files) and
> directory for compiled files (.o and .ppu)?
>
*.bak files are always in the same directory as the source files, 
set 'Project'-'Options'-'Editor'-'Backup File Count' to 0 if you don't want 
backupfiles.
The output directory for  *.o and *.ppu can be set by the -FUxxx compiler 
parameter.
http://www.freepascal.org/docs-html/user/usersu14.html#x37-440005.1.3
