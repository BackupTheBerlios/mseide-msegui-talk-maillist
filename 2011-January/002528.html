"form2pas.prj" builds well if

${MSEDIR}=D:\Program Files\MSEide+MSEgui\ via IDE configure

but can't find MSEgui units if ${MSEDIR} passed via macro:

"D:\Program Files\MSEide+MSEgui\apps\ide\mseide.exe"  
'--macrodef=MSEDIR,D:\Program Files\MSEide+MSEgui\

or

"D:\Program Files\MSEide+MSEgui\apps\ide\mseide.exe"  
'--macrodef=MSEDIR,"D:\Program Files\MSEide+MSEgui\"

Possily because the macro parser should do UNC-vc-Win path convertions.
