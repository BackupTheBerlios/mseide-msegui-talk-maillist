Maybe you should try Github (www.github.com). An account is free and
takes under a minute to setup, access to a new repository is instant -
no waiting or email confirmations. It has excellent features, project
web access, bug reporting, allows others to contribute without you
having to fiddle with permissions (they fork your project, create new
feature in a branch, and send you a pull request).

I've used Github for over a year now and hosting about 5 projects on it
- it works great!

Apparently another good one is Gitorious, but I haven't tried them
myself. But high profile projects like Qt (Nokia) is using them, so they
can't be bad.
