"Inno Setup" seems to be suitable - it supports internal Pascal script  
languages & provides install-time events & dialogues & messages. The below  
is an instance of its code ( checks whether FPC works, via compiling an  
empty program):

[code]
function InitializeSetup: boolean;
var
   ResultCode: Integer;
begin
  result:= true;
  if not Exec(
    'fpc.exe', 'fixes\tests\test.pas', ExpandConstant('{tmp}'),
     SW_HIDE,ewWaitUntilTerminated, ResultCode
  ) then begin
    result:= false;
    MsgBox(
      'FreePascal compiler 2.4+ is not installed but required!' + #13#10 +
      'You may take its win-32 installer at '
      'Cancelling installation...',
      mbInformation, mb_Ok
    );
  end;
end;
