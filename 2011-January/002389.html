The features existing in this installer :
- add files and folder to installer data
- create shortcut/symlink to desktop and programs menu
- change image of installation
- add/modify installation description/other to others language (create
with tstatfile and run in installer builder only)
- compress installation data to zip with tzipper from FPC.

To do :
- encrypt installation text/configuration so end users cannot change.
- add files to execute before/after installation
- add/create/modify files such as create MSEide configuration, so user
doen't need to setting MSEide after installation.

I don't test installer to debian, but I think it can run on any Linux distro.
