For MSEide Installer I would recommend you install the templates in the
users $HOME/.mseide/ directory - or something like that.

For example on my system I startup MSEide with a parameter defining a
new ${TemplateDir} location:

  /home/graemeg/.mseide/templates/

This way you can install MSEide executable in a read-only area, but the
templates and *.sdef files in a read-write area for the user.
