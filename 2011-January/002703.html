> Linux?? I thin GTK use 72 PPI because AFAIK GTK use libcairo for drawing lib.
I try with this command in Linux.

xdpyinfo |grep dimensions

I got :
  dimensions:    1280x800 pixels (338x211 millimeters)

If I calculate to PPMM :

1280/338.00	= 3.7869822485 (38 DOT/CM)
800/211.00	= 3.7914691943 (38 DOT/CM)

So, default PPI (DPI) in Linux is 96 DPI.
