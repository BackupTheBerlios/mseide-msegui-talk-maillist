Brian Winfrey wrote:

> What is the meaning of Settings | Configure MSEIDE:
> ${FPCDIR} (hint: root directory of free pascal system)
> ${FPCLIBDIR} (hint: root directory of FPC library)
> 
Not used in the default MSEide project templates.
${FPCDIR} can be used for example in 'Project'-'Options'-Debugger'-'Source
directories' as root directory macro, ${FPCLIBDIR}
in 'Project'-'Options'-'Make'-'Directories'.
The macros are listed in 'Settings'-'Configure MSEide' in order to allow to
be defined globally.
