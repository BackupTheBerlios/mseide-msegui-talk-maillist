Definitely not for me. I used CamelCase some years before I switched to all 
lowercase. For me all lowercase is much better readable than CamelCase 
because in CamelCase a symbol breaks into several pieces which disturbs the 
symbol recognition. Granted, CamelCase looks better, but I don't need code 
which looks good but code I can read fast, easy and reliable.
