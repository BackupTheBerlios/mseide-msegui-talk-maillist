On Monday, 29. November 2010 09.33:19 Christian Nobel wrote:
> Hi.
>
> Is there a way to ignore components in the tab order - I have an input
> form where the user normally only shall fill in a few fields:
> tab,tab,tab,tab, enter
> tab,tab,tab,tab, enter
> .
> .
>
> However sometimes the user needs to go to some some the other fields on
> the form, which can be done by the mouse, but normally these shall be
> ignored with regards to tabbing.
>
> Any ideas?
>
Disable twidget.optionswidget.ow_tabfocus or change the taborder that the 
often used fields are first. RighClick-'Set Tab Order' can be used to set the 
taborder in a comfortable way.
