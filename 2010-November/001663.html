Graeme,
I adjusted the script and it works fine for me.

Look:
-----------------------------------------------------------------------
@echo on

set myroot=%cd%
set myFPC=%myroot%\fpc\2.4.3
set PATH=%myFPC%\bin\i386-win32;%PATH%

cd mseide-msegui
set myMSE=%cd%
cd %myMSE%\units\i386-win32
del *.ppu
del *.o
del *.a

cd %myMSE%

ppc386.exe apps\ide\mseide.pas -Fulib\common\* -B
    -Fulib\common\kernel\i386-win32 -Fi\lib\common\kernel
    -FE%myMSE% -FUunits\i386-win32 -omseide.exe
-----------------------------------------------------------------------

All ppu were created in units\i386-win32 and mseide.exe was created in
MSE root folder.
