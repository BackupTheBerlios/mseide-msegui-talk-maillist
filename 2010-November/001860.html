The steps:
- Compile with -gh in order the memory of destroyed object instances will be
filled with garbage so the watchpoint will recognize a destroyed object.
- Set a breakpoint into the the constructor of the form which fails.
- Run the program.
- When the program stops at the breakpoint add "fifiserverintf" to watches
(debug1.png).
- RightClick on "fifiserverintf", LeftClick' Address Watchpoint 32'
(debug2.png).
- A watchpoint for the memory address of fifiserverintf will be added
(debug3.png).
- Run the program with F9, the program stops if the memory of fifiserverintf
has been written (debug4.png).
