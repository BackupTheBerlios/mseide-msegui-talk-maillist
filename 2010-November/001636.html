>> Each project has a particular file to highlighting?
>
>
> The files defining the syntax highlighting is normally located (but not
> limited too) the apps/ide/syntaxdefs/
> Accessible via the ${SYNTAXDEFSDIR} macro. Option is in  "Settings >
> Configure MSEide".
>
> Then in each project you specific what file extensions use what syntax
> highlighters. This is available in "Project > Options > Editor tab > Syntax
> Definition Files tab". It is in this screen where you can specify other
> directory locations, other than the global one, for more custom *.sdef
> files, and what file extensions uses what syntax files. If the file
> extension is not listed, no syntax highlighting occurs for those files.
