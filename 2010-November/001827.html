'Save' stores the values of 'Project'-'Options' in the 'Settings' *.prj 
file. 'Load' loads the values with exception of the 'Edit' values 
from 'Settings' *.prj file.

There are no changes made in MSEide project loading/saving.
Example:
You have setup a project based on a template with your locale edit 
preferences.
If you want to publish the settings of 'Project'-'Options'-' 
Debugger', 'Make', 'Macros', 'Font Alias', 'User Colors', 'Templates' 
and 'Tools' enter the name of the export file into 'Project'-'Settings' and 
click 'Save'.
If a user wants to import the settings 
of 'Project'-'Options'- 'Debugger', 'Make', 'Macros', 'Font Alias', 'User 
Colors', 'Templates' and 'Tools' into a project he/she selects the export 
file in 'Project'-'Settings' and clicks 'Load'.

So one allways works with the local *.prj file.
