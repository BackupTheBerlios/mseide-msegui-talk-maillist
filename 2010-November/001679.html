The expanded value in popup hints are very useful, but probably new users
to MSEide doesn't know about this feature. eg: Lazarus and Delphi doesn't
have it in their IDE's. I think it's a very clever feature though.

In my case, since I experienced the problem with a misspelled macro, I now
always check the values via the popup hint - I learn from my mistakes. So I
guess the popup hints are enough, but this feature might be worth
mentioning in the readme file too, so end-users know that they should
double check entered values using the hint window.

As for missing popup hints, I have a few that could be improved on.

In the IDE Settings dialog. All the edit fields with "..." buttons don't
resolve the macro in the hint. The hint simply explains what that setting
does, but I think it will be very useful if the expanded macro value is
displayed in that hint window too (like it does everywhere else in Project
Options for example).
