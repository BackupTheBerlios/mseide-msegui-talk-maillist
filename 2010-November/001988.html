We need to distinguish the old deprecated and from FPC inherited 
TMySQL40Connection, TMySQL41Connection, TMySQL50Connection and the new 
tmsemysqlconnection. tmysqlconnection works with all versions from 5.0 on, it 
does not work with 4.0 and 4.1. The old connection components work with a 
single version only. The FPC connection components link to the FPC files 
mysql40dyn.pp, mysql41dyn.pp and mysql50dyn.pp. In this units there are the 
procedures
"
Function InitialiseMysql(Const LibraryName : String; argc:cint = -1; 
argv:PPchar = nil; groups:PPchar = nil) : Integer;
Function InitialiseMysql(argc:cint = -1; argv:PPchar = nil; groups:PPchar = 
nil) : Integer;
Procedure ReleaseMysql;
"
where a specific dll can be preloaded.

MSEgui tmysqlconnection links to the unified MSEgui unit mysqldyn.pp with the 
procedures
"
procedure initializemysql(const sonames: array of filenamety); 
                                            //[] = default
procedure releasemysql;
"
for the purpose.
