Marcos Douglas wrote:

> How I can compile the MSEide and put the .ppu and .o files organized
> like \units\i386-win32 ?

Please use the -FE compiler parameter.
http://www.freepascal.org/docs-html/user/usersu14.html#x37-440005.1.3

Compiler parameters can be set in 'Project'-'Options'-'Make'-'Make options'.
