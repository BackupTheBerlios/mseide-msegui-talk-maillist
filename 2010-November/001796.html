Hi Martin,

It's only come to my attention now that the binary downloads of MSEide
doesn't ever contain any syntax highlighters or a simple project
template or two (eg: a console project with a few platform macros
etc). I think these should at least be included with the binary. They
will hardly add any extra download size, and will at least make the
IDE useful to those that simply want to use it as an editor and
debugger with FPC.

PS:
There was just such a request in the FPC-Users mailing list a few
minutes ago. They simply wanted a binary IDE with GDB debugger
support.

I suggested MSEide, but then saw it doesn't even have the templates or
*.sdef files, which makes the binary rather useless (or much much
harder to use).  Maybe you can update the 2.4 binary release downloads
with these missing bits? It will add a lot of value to the binary
downloads.
