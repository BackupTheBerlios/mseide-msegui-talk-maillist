On 07/14/2011 10:54 AM, Martin Schreiber wrote:
>
> Personally I don't like tabs in source code, I use 1-space indentation.

1-space indentation is just too little of an indentation for my liking.
Personal preference I guess. The good thing with ET is that you can
adjust the "global" indentation setting as you need, and all your source
code adjusts without modification. So the same source code files can
look different (regarding indentation) depending on each developers
preference settings.

The important point being that ET doesn't insert more TAB characters to
align text (like standard tab indentation does), it instead
shifts/adjusts the indentation value per block of code, but still using
a single TAB character.
