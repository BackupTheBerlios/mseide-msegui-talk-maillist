On Saturday 16 July 2011 05:24:05 wahono sri wrote:
> Any examples/demo how to use topenglwidget?
>
topengelwidget manages an OpenGL render context. The following event handlers 
draw a color quad:
"
procedure tmainfo.clientrectchangedexe(const sender: tcustomwindowwidget);
begin
 glMatrixMode(GL_PROJECTION);
 glLoadIdentity();
 gluPerspective(45,1,0.1,200);
 glMatrixMode(GL_MODELVIEW);
 glLoadIdentity();
 glTranslatef(-0.5,-0.5,-2);
end;

procedure tmainfo.renderexe(const sender: tcustomopenglwidget;
               const aupdaterect: rectty);
begin
 glClear(GL_COLOR_BUFFER_BIT);
 glBegin(GL_QUADS);
  glColor3f(1,0,0);
  glVertex3f(0,0,0);
  glColor3f(0,1,0);
  glVertex3f(1,0,0);
  glColor3f(0,0,1);
  glVertex3f(1,1,0);
  glColor3f(1,1,1);
  glVertex3f(0,1,0);
 glEnd();
end;

"
see attachment.
I started using OpenGL as additional MSEgui render engine. topenglcanvaswidget 
has a MSEgui canvas with OpenGL rendering backend (not finished!). Compile 
MSEide with -dmse_with_openglcanvas in order to install the component.
