It is not exciting, the purpose is to have an additional render backend for 
MSEgui appart from X11 and gdi32. Build MSEide with mse_with_openglcanvas in 
order to install topenglcanvaswidget.
If you want to play with OpenGL use topenglwidget. An example is here:
http://svn.berlios.de/viewvc/mseuniverse/attic/msedocumenting/mse/trunk/help/tutorials/widgets/opengl/openglwidget/
