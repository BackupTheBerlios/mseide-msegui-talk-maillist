It's just a little bit more complicated, as Linux uses a rather strict
library reference system, as you found. But there is a way around provided
as well: You have to set up the environment variable "LD_LIBRARY_PATH" to
contain the path to the directory where the library resides. If you use an
expression like "dirname $0" in a wrapper shell script, that lets you put
any ancilliary libraries in the program directory and still have them
available for your program.
