On Monday 04 July 2011 04:25:14 wahono sri wrote:
> Is that to change current application project (application.exe) or to
> MSEide?
> I try to create alias : stf_default, font name : arial and size : 6, but
> the IDE font changed.
>
The purpose is is to define runtime custom font alias at design time, 
comparable with 'Project'-'Options'-'User Colors'. At run time use 
registerfontalias() function from unit msefont in order to create a custom 
font alias. Please use the command line parameter --FONTALIAS= in order to 
override a registered font alias at runtime.
