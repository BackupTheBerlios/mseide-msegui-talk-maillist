Using trealdis  you can format the output. If you want to display only two
decimals, select your trealdis widget and in object inspector set the
property 'format' to 0.00  this will render only two decimals.

Keep in mind, in this way you aren't limiting the number of decimals (you
are only limiting the number of decimals shown). It depends of the real type
used. You can take a look here:
http://www.freepascal.org/docs-html/ref/refsu6.html#x28-310003.1.2  and for
float formating, here:
http://www.freepascal.org/docs-html/rtl/sysutils/formatfloat.html

Cheers.
