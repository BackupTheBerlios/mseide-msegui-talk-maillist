Once my programs are working fine, is time to set it beautiful. Skins,
faces... are great. I'm happy using it. Icons are another thing... default
icons using in MSEide+MSegui look a bit outdated to these ages (I say it in
a friendly way) :)

In my applications I use silks icons (
http://www.famfamfam.com/lab/icons/silk/ ) they are pretty nice and I find
almost every thing I need (for buttons icons)

In example, I change the default icons for file dialogs and from my point of
view, I get better results (look at the attached pics)

I wonder if is there a way to change the default glyps icons (other than
modifying MSE* sources) like I do for file dialogs. Probably you also can
consider using fam-fam icons (silk) the author only requires you to provide
