On Monday 07 March 2011 19:55:22 Julio Jiménez wrote:
> Hi, I have observed that changing font size modify some widgets height, but
> no others, i.e. tstringedit and tbutton. Probably this is as designed but,
> why tbutton is not growing as other widgets do when using different fonts
> heights? I can change the height in the program but I think is not the most
> correct.
>
> Is there a way to make tbutton working like tstringedit and others?
>
Activate optionswidget ow_autoscale and ow_fontglyphheight. Should be on by 
default in tstringedit. Or activate optionswidget1 ow1_autowidth, 
ow1_autoheight.

Martin
