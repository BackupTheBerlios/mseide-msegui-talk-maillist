On Wednesday 23 March 2011 18:42:18 Sri Wahono wrote:
> Is possible to set maxlength on tdbstring*edit automatically, which
> the value is get from tstringeditwidgetdatalink?
>
The stringlength is automatically limited by tbufdataset:
"
procedure tmsebufdataset.setmsestringdata(const sender: tmsestringfield;
               const avalue: msestring);
var
 po1: pointer;
 int1: integer;
begin
 sender.validate(@avalue);
 po1:= getfieldbuffer(sender,false,int1);
 msestring(po1^):= avalue;
 if (sender.characterlength > 0) and 
                     (length(avalue) > sender.characterlength) then begin
  setlength(msestring(po1^),sender.characterlength);
 end;
 fieldchanged(sender);
end;
"
