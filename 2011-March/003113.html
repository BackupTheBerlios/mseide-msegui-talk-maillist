File" dialog) in MSEide must still appear in the center of the screen,

that same code to "Project > Open" dialog.

by side). It seems some applications can detect I am using multiple 
monitors, and if a dialog should appear in the center of screen, they 
use the monitor in which the main window of that application is in as 
the "screen", instead of my whole desktop area. I don't know how they

was easier (and quicker) to implement, than trying to figure out Linux
