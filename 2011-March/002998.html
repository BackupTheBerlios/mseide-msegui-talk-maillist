:-) Maybe not cheaper, but LCD monitors are really not that expensive
these days.  Either way, I have used a 4x virtual desktop setup since
the day I started using Linux (15+ years ago). It's quite amazing that
Windows (I don't know about Win7) doesn't come standard with virtual
desktop support - even MacOS has virtual desktops now.

Even though I use two monitors, I still use a 4x virtual desktop. I've
learning from my OS/2 days, that keeping most used apps open all day is
much quicker that start-stop method. I run *lots* of apps concurrently.
With a virtual desktop only setup, you still have overlapping windows on
a single desktop - that's where a multi-monitor setup rocks (especially
in debugging apps)!
