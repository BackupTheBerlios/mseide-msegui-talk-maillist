Or mount the Windows partition on "/c" and install MSEgui on Windows in
"c:\home\user\opt\mse\msegui" then 
"c:\home\user\opt\mse\msegui\trunk\lib\common\kernel\mseglob.pas"
will become
"/c/home/user/opt/mse/msegui/trunk/lib/common/kernel/mseglob.pas"
on Linux which are the same paths in MSEgui.
