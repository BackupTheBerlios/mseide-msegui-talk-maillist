I fully agree. The state of "a graphical desktop" under Linux is
absolutely appalling. The worst performance of *any* desktop OS I have
ever tried. Even OS/2 Warp v3 had better performance.

It seems they think that adding compiz and wobble effects will detract
the users of the truth. Very sad, and considering I'm using the

year old laptop). It only used the VESA drivers, yet graphics was very
fast. Resizing windows while content is being displayed has near zero
cursor-to-window edge lag. Again just proving what a sad state the Linux
desktop graphics is in - even a non-complete OS already has better
graphics performance than Linux. Lets hope X11 dies really soon, and
something better takes it's place.
