The projectdir is used to relocate files. Files are first searched in absolute 
path, if not found they are searched relative to projectdir. Please use 
macros for your directory definitions which can be switched easy 
in 'Project'-'Macros'.

> 3) Some paths may disclose confidential information (e.g. lastdir) !

Please use 'Project'-'Settings' 'Save' in order to store the project settings 
without window layout and the like. 'Editor' defines if the editor settings 
are stored or not.
