On Tuesday 22 March 2011 12:45:43 Smile Girl wrote:
>  Hi,
>
>      I have to create a simple form from main form.
> Can i send data from main form ?
>
> Example:
> If there r three tasks, at the end of each task I create a simple form
> with some data to be put to simple form..
>
> Suppose in simple form I have a string display.. Can I call the same
> form to display data from different tasks..

Yes. Add the mainform unit to "uses" of the "simple forms" then the components 
of the mainform instance (usually "mainfo.xxxx") are accessible.
If the "tasks" do not run in main thread, application.lock()/unlock() must be 
called for accessing GUI elements.
