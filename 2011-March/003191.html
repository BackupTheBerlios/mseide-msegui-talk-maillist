This file is not relative to the project directory so it can not be relocated. 
The projectdir entry in the prj file actually will be used for file 
relocating as you see in the path above.
"
  /media/c/test_p/main.pas
  /media/c/test_p/test2.pas
  /media/c/test_p/subf/test3.pas
"
were relocated.

If you place the MSEgui source tree in all OS at the same place relative to 
the project directory relocating will work. If you can't do so I suggest to 
use different *.prj in the different OS (myproject_win.prj, 
myproject_lin.prj). There can be a common myproject.prj which can be 
synchronized with 'Project'-'Settings'+'Load' or 'Save'.
The 'File not found' error is not dangerous BTW, simply click OK.
