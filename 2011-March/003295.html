The world is going HW acceleration crazy. Not having hardware 
acceleration for drawing a few things on the screen is hardly a problem. 
Yes if you have some 3D CAD system or a 3D game engine, then yes it is a 
issue, but for standard 2D desktop apps, I think not. The AggPas demos 
run perfectly smooth on my system. The application is short-lived, so

the demo. Not to mention that normally such apps only push one of my 
four cores to 100%, with the others three cores still at idle - so I 
notice no slowdown.
