The tdataimage is not released but the assigned pixmap.

> > Please use tmaskedbitmap.writetostring(). The necessary image format
> > units (formatpngwrite, formatjpgwrite...) must be listed in uses.

> 
> picture.bitmap.writetostring(<ansistring>);
>

Because the definition is:
"
   procedure writetostring(out avalue: string; const format: string;
                                  const params: array of const); overload;
   function writetostring(const format: string;
                           const params: array of const): string; overload;
"
Please use Ctrl+LeftClicke or Shift+Ctrl+Space after "(".
