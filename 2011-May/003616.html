Am 15.05.2011 23:29, schrieb Patrick Goupell:
> Is it possible to set the scroll position in a stringgrid?
>
> I have a stringgrid loaded from a mysql database.
>
> The user scrolls to a row within the grid.
>
> At this point the user can add, change or delete a row (record).
>
> If the user requests a refresh of the stringgrid I want to reposition the scroll
> position to what it was at the time of the refresh request.  Meaning if the user
> is in the middle of the stringgrid (item 100 of 200 visible) I want the
> stringgrid to again be visible in the middle.
>
Please use tcustomgrid.showcell():
"
    procedure showcell(const cell: gridcoordty;
                       const position: cellpositionty = cep_nearest;
                       const force: boolean = false);
       //scrolls cell into view, force true -> if scrollbar clicked also
"
