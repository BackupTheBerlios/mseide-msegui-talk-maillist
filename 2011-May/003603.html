> If I have an stringedit in a widgetgrid, I can insert a value like this:
>
> tstringedit2.gridvalue[y]:='blablabla'
>
> How do I do something similar if it is a picture?
>
> In a dbwidgetgrid I can do this:
>
> TBlobField(tmsedatasource1.dataset.FieldByName('thumb')).LoadFromStream(sma
>llpicstream);
>
> So my assumption would be like this:
> 				tdataimage1.gridvalue[1].bitmap=smallpicture;

> tdataimage1.gridvalue[1]=smallpicture;
"
 tdataimage1.gridvalue[1]:= <yourimagedataasansistring>;
"
ex.:
"
 tdataimage1.gridvalue[1]:= readfiledatastring(<yourimagefilename>);
"
The necessary graphic format units (ex. "mseformatpngread") must be listed in 
uses.
If the same small image should be displayed in several rows it is better to 
use tdataicon and a timagelist.
