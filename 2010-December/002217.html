Du schriebst am Thu, 23 Dec 2010 09:06:03 -0800:

> What is the equivalent of align in msegui?
> VCL/LCL align can be set as alClient, alTop etc. and have
> margin/padding setings as well.

MSEGUI uses the anchors for thet purpose; Delphi's "alClient" is achieved
by clearing all anchors, "alTop" is equvalent to setting only the tiop
anchor and pinning the components at bounds_y = 0, and so on.
In general, clearing one set of anchors (horizontal / vertical) makes the
component stretch out over all of the available space.
Margins or padding, BTW, can be set in the accompanying frame subcomponent.
