On Tuesday, 14. December 2010 22.01:00 Coyot.Rush wrote:
> how to convert Tbitmap (MseGui) in Tpixmap (XLib)?

MSEgui tbitmap has two representations of the image. Normally it is a server 
side X pixmap where the pixmap ID is stored in tbitmap.handle, pixmapty is a 
pixmap X-id on Linux. For tmaskedbitmap there can be an additional monochrome 
or color pixmap handle for the mask (tmaskedbitmap.mask.handle). By accessing 
tbitmap.scanline the server side pixmap will be converted into an internal 
memory array of colorty for a color pixmap or an array of byte for monochrome 
bitmap.
