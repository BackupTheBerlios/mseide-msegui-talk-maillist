> What app should I install to create window manager?

If you really want to stay with the Free Pascal theme, then why not
try the fpGUI Window Manager. :) The project is on SourceForge, but I
don't think anybody has worked on it in a long time.  This will all be
experimental though.

If you want to go with tried and tested, yet still light weight you
can used Joe's Window Manager (JWM). I use this myself and it is
really good and fast, and easy to customize (eg: like removing the
taskbar for a POS system). You also can't go wrong with the FVWM
window manager. In fact there are really a lot of window managers to
choose from, but I know JWM and FVWM very well, and they work
fantastic.
