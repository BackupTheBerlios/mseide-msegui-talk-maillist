Am 11.12.2010 09:46, schrieb dan:
> Hi, Martin. I have two questions:
> 1. How to use ns_checkbox(activated) in ttreeedititem? why checkbox
> isn't appeared?

ns_checkbox must be activated in the individual node states.

> 2. I use tbooleanedit, ttreeedititem, tstringedit in twidgetedit. And
> i have some data in that twidgetedit.
>   When i collapse and then expand one of tree in ttreeedititem, value
> in tbooleanedit is blank(missing).

ttreeitemedit maps expanded tree nodes to grid rows, there are no rows 
for collapsed nodes. The values of additionally columns must be 
transferred from treeitem in code, for example in 
ttreeitemedit.onupdaterowvalues.
If you want to have rows for all items without using treeitem use a 
tfoldedit instead ttreeitemedit. optionsgrid og_folded must be activated 
in order to enable folding support. The folding state of the rows are in 
the rowhidden/rowfold* properties.
