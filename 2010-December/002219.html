> I've download fpGUI, where and how to use fpGUI window manager?

It's a separate project on SourceForge... I think it was called
'fpwm'.  As I said, it is very experimental, and was created with a
old fpGUI (from around 2-3 years ago).


> I use OpenSuse with package textmode (without any window manager). How
> to activate a window manager from console?

A google search will find that info for you, for your specific distro.
Not all distros use the same configuration files (another lovely mess
of Linux). In a pinch, simply start X with a terminal window, then
simply run the window manager as a background application:

eg:

   $  jwm &

Then run your application. Window Managers are simply X11 application
with a bit more rights than normal desktop apps.
