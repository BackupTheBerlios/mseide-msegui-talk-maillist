> Please use tmsetransaction.Params property.

In tmsemysqlconnection, I can't add command SET GLOBAL TRANSACTION
ISOLATION LEVEL READ COMMITTED to tmsesqltransaction.params, because
of tmsesqltransaction.params is processed after command START
TRANSACTION params.

Than I add the command on tmsemysqlconnection.afterconnect event with :
tmsemysqlconnection.executedirect('SET GLOBAL TRANSACTION ISOLATION
LEVEL READ COMMITTED'); and It result as I want.

I don't test it on firebird, postgresql and sqlite.
