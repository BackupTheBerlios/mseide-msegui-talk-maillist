> how to execut a  sql procedure with output and get the output

For normal DB backends supporting SELECT for procedures:

tmsesqlquery/tsqlresult.sql.add('select proc_name(ARGS,...)');
tmsesqlquery/tsqlresult.Active:= true;

POSTGRESQL also supports IN/OUT args for procedures - OUTs will be  
returned as a table of fields in the order of OUTs.
As the result set can aslo be a table.
