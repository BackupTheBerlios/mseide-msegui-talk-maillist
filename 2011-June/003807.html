Please note: after editing, the first return keypress is used to enter the 
value, the next return keypress will be handled by the default button. If 
optionsedit oe_eatreturn is disabled, the first return keypress will not be 
eaten if the value has been edited.
Please send a simple test project if it does not work for you as described.
