2011/6/1 Julio Jiménez <jujibo at gmail.com>:
> There are may ways to do it. By default Freepascal like Delphi, use Banker
> rounding system (round half to even) In my country this isn't valid. We have
> to round half up like you say.
>
> You can google for lots of examples.
>
> A way to do it:
>
> function RoundD(x: Extended; d: Integer): Extended;
> // RoundD(123.456, 0) = 123.00
> // RoundD(123.456, 2) = 123.46
> // RoundD(123456, -3) = 123000
> var
>   n: Extended;
> begin
>   n := IntPower(10, d);
>   x := x * n;
>   Result := (Int(x) + Int(Frac(x) * 2)) / n;
> end;
>
