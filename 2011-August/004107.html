No idea, but Martin should be able to answer that. I can't imagine
that MESifi would come close to the features of CORBA. I am working
with the author of mtdORB to get it working with Free Pascal though.
Apparently the missing units I mentioned before, are not really
missing. You need to generate them from the *.idl (corba interface)
files. I have created a script which does this correctly, because they
need to be generate with specific parameters. mtdORB only supported
this via some ANT build system. I have also fixed a whole bunch of FPC
compiler errors so far - due to mtdORB  not having been testing with
the last few FPC releases.

As soon as I'm done with all the changes, I'll supply patches back to
the mtdORB project. The current mtdORb author is very happy to apply
such patches to get FPC supported again. As soon as I have submitted
the patches I'll post a note here, so you should be able to get a CVS
update shortly after that.
