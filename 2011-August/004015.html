> Can I use M$ Word for these templates?

Yes. You can use any editor that can save files as RTF. You can also
use any editor to view the generated report, which is in RTF too. But
like I mentioned, not all editors make greate "viewers". eg: MS Office
is terrrible, because even though you launch it in read-only more (or
mark the RTF file as read-only), MS Office still shows all editing
function, and even allows you to edit the document! Hence we use
OpenOffice, and we can ship in freely with our products. You can even
use the built-in macros of OpenOffice in a non-gui mode (no UI or
splash screen is ever shown), to generate PDF files, so the end-user
will never know about the RTF files.
