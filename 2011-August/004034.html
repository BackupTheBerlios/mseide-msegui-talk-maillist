> Once you have DB support via tiOPF, it is irrelevant what backend you
> use. Any DB demo will work with any DB backend. Simply toggle the
> backend via any one of the LINK_xxx compiler defines. See the
> tiOPFManager.pas unit in the uses clause of the implementation section,
> to get a listing of all available compiler defines. There are many!
> SQLite3 is currently supported via the SqlDB database components
> (accessed via the LINK_SQLDB_SQLite3 compiler define).
