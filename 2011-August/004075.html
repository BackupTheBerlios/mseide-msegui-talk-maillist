I'm very interested in CORBA as well - I think it's an excellent
designed, and much better thought-out than WebServices. Unfortunately I
haven't had any real-world work with it yet.

Anyway, there is mtdORB which is a fully compliant implementation of
CORBA 2.3 standard. It was initially written for Delphi, then Kylix and
FPC support was added. The FPC support is basically using the FPC Kylix
compatibility unit 'libc.pp', thus Linux support is limited to 32-bit
only. I don't know if work has been done to make it fully Linux, FreeBSD
etc compatible by using the FPC baseunix units instead. Latest code is
in SourceForge in the CVS tree.

  http://sourceforge.net/projects/mtdorb/
