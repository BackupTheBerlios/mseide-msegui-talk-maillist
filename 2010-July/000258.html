> 2010/7/2 Martin Schreiber

>> Use a tsequencelink (tab DBf) in order to automatically fetch the value
>> from the sequence generator and store into the "id" field before
>> inserting a record.
> 
> This good idea because may need the fresh id other reasons. For
> example insert other child record to a details table.
> But how can I use this nextval in the insert statement?
> I tried SQLInsert property but don't made any parameter from ":value"
> like in query does.
> 
- Place a tsequencelink in the form or datamodule.
- Enter the sequencename.
- Select database.
- Select datasource and datafield (the "id"-field).

Now the id field will be set with nextval('<sequencename>') by TDataset.Post
after insert.
