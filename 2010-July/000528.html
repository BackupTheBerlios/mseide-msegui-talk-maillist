Christian Nobel wrote:

> Hi again.
> 
> Additional Q.
> 
> In a tstringgrid I can get the value of a cell like this
> 
> blabla:=tstringgrid1[0][0];
> 
> However I can of course not du the same with a twidgetgrid, as it
> returns a widget.
> 
> I can do like this:
> 
> blabla:=tdropdownlistedit1.gridvalue[0];
> 
> where tdropdownlistedit1 is the first column in the widgetgrid.
> 
> But is there a way to addres the columns directly and then get the
> contents of the widget, without explicit using the name of the widget?
> 
It is not possible to access the grid values directly by the column because
the type is variable. The datawidget of a column is in property
twidgetcol.editwidget. The griddata can also be accessed by
tdatacol.datalist.
