Perhaps a button too. And onexecute event handler of this button should call the 
test code.

You can make room for code the following:
1: drop the button onto form (by selecting from component palette widget tab)
2: on object inspector you can find "onexecute" field.
3: type into that a proc name eg: exec1, then press enter key.
The code editor make a declaration and an empty definition for you, just type 
the body of code of event handler. Between the begin and end keywords.
On runtime this code will fire up when you press the button.
