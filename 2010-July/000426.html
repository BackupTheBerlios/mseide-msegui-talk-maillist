Christian Nobel wrote:

> Hi.
> 
> Have a small problem, which puzzles me.
> 
> In a project where I am using a twidgetgrid, I want to apply captions to
> the fields, when the program starts like:
> 
> WGridWhere.fixrows.rows[-1].captions[4].caption:= 'Værdi';
> 
> This works, but the Danish character æ is not displayed correctly at
> runtime.
> 
> If I cut and paste 'Værdi' into the Object Inspector - fixrows.count -
> Item-1 - captions.count - Item4, then the value is show correctly at
> runtime.
> 
Add -Fcutf8 to the compiler options so that FPC knows the source is utf-8
encoded, otherwise ISO8859-1 is assumed. Set 'Options'-'Editor'-'Encoding'
to 'utf8' in order the source files are stored in utf-8 on Windows.
