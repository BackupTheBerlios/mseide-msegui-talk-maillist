The rounded widget frames are made with tframe.frameimage_*. In
frameimage_list are 8 images which are used in order to draw the frame
counter clock wise. The image N is placed in the topleft corner, N+1
stretched for left edge, N+2 bottomleft corner, N+3 stretched for bottom
edge, N+4 bottom right corner, N+5 stretched for right edge, N+6 topleft
corner, N+7 stretched for top edge. There are different starting offsets in
the imagelist for different widget states. 
The centre of the frame can be filled by tframe.frameface_list. There are
different entries for the face number in tfacelist for different widget
states.
Because every widget in MSEgui has a frame and face property every widget
has these possibilities.
Use tfacecomp and tframecomp and tskincontroller in order to centralise
skinning. tframecomp can be selected as a template into tframe.template,
tfacecomp can be selected as a template into tface.template.
