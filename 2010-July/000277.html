Normally in the main program:
"
program test;
{$ifdef FPC}{$mode objfpc}{$h+}{$INTERFACES CORBA}{$endif}
{$ifdef FPC}
 {$ifdef mswindows}{$apptype gui}{$endif}
{$endif}
uses
 {$ifdef FPC}{$ifdef linux}cthreads,{$endif}{$endif}msegui,mseforms,
           main,datamod;
begin
 application.createdatamodule(tdatamodmo,datamodmo);
 application.createform(tmainfo,mainfo);
 application.run;
end.
"
Datamodules which have component links to other datamodules or forms must be
created by application.createdatamodule, forms which have component links
to other datamodules or forms by application.createform in order to be
registered for inter module component resolving.
