Uhh, sorry I was not at work yesterday. The code I posted
"
 tcustomgrid = class [...]
   procedure moverow(const curindex,newindex: integer; const count: integer
= 1);
   procedure insertrow(aindex: integer; acount: integer = 1); virtual;
   procedure deleterow(aindex: integer; acount: integer = 1); virtual;
"
Was copied from msegrids.pas, tcustomgrid is the base class for all grids ->
all grids have this procedures. In order to see the available procedures of
a class Ctrl+LeftClick on the type of the definition (twidgetgrid in
example).
"
type
 tsimplefo = class(tmseform)
   twidgetgrid1: twidgetgrid;
 end;
"
twidgetgrid will be displayed. Ctrl+LeftClick on tcustomwidgetgrid in order
to show the ancestor tcustomwidgetgrid, Ctrl+LeftClick on tcustomgrid in
order to show the ancestor tcustomgrid.
Scroll down until the public procedures, functions and properties are
visible.
Navigation back and forth with the blue arrow buttons in topright corner of
the source editor.
