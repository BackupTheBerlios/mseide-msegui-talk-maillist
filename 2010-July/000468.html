Christian Nobel wrote:

> Hi.
> 
> Is there a way to a insert a row in a twidgetgrid, and also a way to
> delete a row, not just setting the rowcount to rowcount-1 but moving up
> the rows following where one wants to delete?
> 
"
 tcustomgrid = class [...]
   procedure moverow(const curindex,newindex: integer; const count: integer
= 1);
   procedure insertrow(aindex: integer; acount: integer = 1); virtual;
   procedure deleterow(aindex: integer; acount: integer = 1); virtual;
"
Shortcut insert = Ctrl+Insert delete = Ctrl+Delete, optionsgrid
og_rowinserting, og_rowdeleting must be enabled.
