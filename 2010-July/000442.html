I just discovered, using recent trunk key down event is only fired 1
time, key up is working as expected.


This snippet of code: (capturing onkeyup and onkeydown

procedure tmainfo.keyupdone(const sender: twidget; var ainfo: keyeventinfoty);
begin
 case ainfo.key of
  key_Insert: writeln('insert up');
  key_F1 : writeln('F1 up');
 end;
end;

procedure tmainfo.keydowndone(const sender: twidget; var ainfo: keyeventinfoty);
begin
 case ainfo.key of
  key_Insert: writeln('insert done');
  key_F1 : writeln('F1 done');
 end;
end;

The result in console after pressing three times F1 and Insert Key:

insert done
insert up
insert up
insert up
F1 up
F1 up
F1 up
