Ivanko B wrote:

>>Trunk 3811 has tsqlquery.SQLInsert/SQLUpdate.options uso_refresh.
>>RETURNING clause used in order to update the field values after executing
>>the statement.
> 
> What's meaning of this option ?

Uses RETURNING for INSERT or UPDATE in order to refresh DB created field
values. Can be used for example to replace sequence generated field values
in applyupdate() instead in post() by tsequencelink.
