And don't forget that if you are not really interested in the whole
history, you can tell git to only fetch x amount of commits for each
branch.

eg:

  git clone --depth 5 git://gitorious.org......

Now it will only fetch the last 5 commits of each branch, making a
clone much quicker to complete and less to download (if you are on
expensive internet). But then again, you only every need to clone a
remote repository once. After that you can use your existing clone to
make other clones on a USB flash drive etc
