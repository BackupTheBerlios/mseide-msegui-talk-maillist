Done, trunk 4852.

- Set <twidgetgrid>.optionsgrid og_rowheight in order to enable support for 
variable grid row height.
- Set <twidgetgrid>.datacols[<thecolofthetterminal>].options1 
co1_autorowheight.
- Set <tterminal>.textflags tf_wordbreak.
