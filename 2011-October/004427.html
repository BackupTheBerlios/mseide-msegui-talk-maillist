BTW:
I'm starting to see your logic about storing editor settings per
project too - instead of global settings like Delphi or Lazarus. I've
switched some of my projects to using TAB character indentation, but
most of my projects are still 2 Space character indentation. Using
MSEide to switch between projects is a breeze - editor settings is
always correct. On the other hand, using Lazarus is a pain - because I
need to keep adjusting my editor settings depending on which project I
am working on.

Your ideas might be "non standard", but they clearly have been given
some thought, and DO work in the real world. Keep up the good work!
