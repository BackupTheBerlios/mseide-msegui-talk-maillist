BTW, This below:

> There is a non-profit organization in the UK developing an ARM based PC

is _not_ a real PC, you even said it above. It is an embedded computer card
using a very non-PCish CPU - ARM - which will _not_ run any regular PC
operating system. You won't be able to run a Windows XP, Vista or 7 on it,
the latest Windows system available for ARM is Windows CE (AFAIK).
You cannot run one of the common Linux distributions on it. You will need a
special ARM version of the system, and although they exist, you most
probabely will have to build and maintain it yourself. You will also have
to build and maintain all applications yourself, and don't count on that
they will port 1:1 to this processor - you might have to do quite some bug
hunting and patching here.

> I think this device has HUGE potential in our Master Maths/Science

Probabely so, it will be an excellent "educational" tool. It might also
make a good small volume / single piece embedded system core. But it will
not be a plug-and-play PC replacement by any means.

> centres. A really small PC with no movable parts, thus very robust. It

You can get "real" PC boards of similar structure (featuring an intel or
AMD processor and other common PC components) also - and these will be far
easier to use for someone not involved with developing a special device, as
they can be set up with most any installation-ready operating system
available.
