The problem is that I am still struggling with getting from
Delphi/Lazarus to MSE, as allmost everything is more or less different.

EG Delphi/Lazarus Memo:

Memo1.Lines.Add('string');

MSE tmemoedit:

tmemoedit1.text:=tmemoedit1.text+'string'+chr(13)+chr(10);

So for each and every time, I am going to use something in MSE, I can
spend half a day to figure out which procedure I can use, and how the
properties are etc.

I can remember that learning Delphi code completion was an enormous
help, as the code completion easy gives one the overview of what can be
used in a given situation.

And I think it is a pity, because if MSE is going to be used by more
than a little elitist group something have to be done.

My feeling is that a lot of Lazarus users finds MSE, and think it looks
promising, as it does take hand of a lot of the caveats in Lazarus, but
they give up, because it is to difficult, and go back to Lazarus.

Please dont understand this as me being negative, as I very much would
like MSE to be a succes.
