Me again...

When I trigger a code template, the IDE seems to add an extra newline
after the template, which does not exist in the *.mct files.

Attached is a simple code template that add's a TODO item inside code
comments. Something I use often in my code.

An example of what happens...


Before the code template:  | symbol denotes the cursor position
------------------------------
uses
  todo|
  Classes, fpg_base, fpg_main, fpg_label;
------------------------------


...and after the code template has been triggered....

------------------------------
uses
  { TODO : | }

  Classes, fpg_base, fpg_main, fpg_label;
------------------------------


...what I expected to see is this....

------------------------------
uses
  { TODO : | }
  Classes, fpg_base, fpg_main, fpg_label;
------------------------------



Now if you look at the attached general_todo.mct file, the actual body
of the template is just a single line, there is no LineEnding
character(s) at the end of the file.


PS:
I'm at 20 code templates already.  Clearly code templates is a feature
I use often without even thinking. :)
