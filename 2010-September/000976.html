tsqlresult works with the RETURNING clause. If you want to use tmsesqlquery
set ParseSQL to false and StatementType to stSelect. It is possible that it
you need to activate the SQL editor and clicking OK afterwards in order to
refresh the settings (bug).
If you want to refresh server generated field values automatically after
applayupdate() in tmsesqlquery activate t*field.providerflags1 pf1_refresh.
