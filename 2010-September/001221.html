It depends if you need the data organized in a tree with possibly different
and complex datatypes in the nodes. In this case use ttreeitemedit.
If the data are simple basic types and the same on all rows tfoldedit can be
used too. There will be no data storage in a tree. Use
tcustomgrid.rowfoldlevel in order to set the fold indentation of a row.
