Graeme Geldenhuys wrote:

> 
> A few issues I found:
> 
> 1) cursorrow seems zero based, but the MSEide and all other editors,
> the first line in a text file as show to the user is line 1, not line
> 0.
> 
corsorrow (same as cursorcol) is the offset from current cursor. MSEgui
ttextedit and tcustomgrid are nullbased too.

> 2) placing the cursor is quite problematic, especially if parameters
> are used. Depending on the length of the parameter value, I can't
> foresee the cursorcol position.

Correct. It is planned to adjust the cursor position by the macro expansion
but it is a little bit complicated...

> line numbers in large templates so I know what line number to use.

I didn't want to do character escapement in case one wants to use '|' in
text. Later there probably will be a tool for editing templates.

> template file.
> 
The intention is that indention can be set easy with Ctrl+I.
