Christian Nobel wrote:

> In Delphi (and Lazarus) cursor is set like this:
> 
> mainfo.Cursor := crHourglass;
> 
> Which mumbo-jumbo is going to be used for doing the same with MSE?
>
I would use application.beginwait, application.endwait. ;-)
Also useful in this context is application.waitescaped, returns true if Esc
was pressed while waiting.
If you want to change the cursor over the main form use mainfo.cursor:=
cr_wait. The names of the cursor shapes are named like the X11 standard
cursors.
