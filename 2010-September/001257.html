Trunk 3953 has experimental support for code templates in MSEide.

Extension for template files is *.mct
Example of a template file:
"
[header]
name=test
comment=Comment abcdefg HIJKLMNO
params=3
 param1
 param2
 param3
[]
The Template Text with macros ${param1}
more text
${param2} ${param3} more params
${msedir} a MSEide macro
"

Minimal template file:
"
[header]
name=test
[]
"
All MSEide macros can be used.
The searched template directories are listed
in 'Project'-'Options'-'Editor'-'Code Templates',

Parameters can be entered in editor:
"
test,valueforparam1,valueforparam2
"

A dialog window will be showed for param entry if there are more params
defined than supplied . Default shortcut is Ctrl+T.
