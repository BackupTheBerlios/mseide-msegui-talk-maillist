Hi everybody,

Attached is a very small sample of MSEgui documentation in INF format
(eg: TMSECustomForm Properties overview, or FormOptionTy enum meaning
etc). I tried to use the documentation from MSEDocumenting, but in was
chaos to find anything (there seems to be no structure - sorry, but
this is true). So just wrote some of my own documentation based on the
names of the properties.

To view the msegui.inf document, you need to use DocView, which is
available for download (as pre-compiled binaries) from SourceForge.
There are Windows and Linux binaries. DocView download is between
350KB and 550KB in size.

  http://sourceforge.net/projects/fpgui/files/

I can supply a complete "skeleton" class documentation of MSEgui in
INF format - if you want. It is considerably larger, but at least one
can search for methods, properties, classes etc. with great speed. So
this might be useful to someone new to  the MSEgui framework.

Hopefully this can serve as an inspiration to all, to try and
contribute to MSEgui documentation, but in a well defined an
structured manner that fpdoc gives use. If the documentation is in
fpdoc format, then various output formats can be generated: HTML, RTF,
PDF, CHM, INF, TXT, MAN etc.  all from one set of source documentation
files.
