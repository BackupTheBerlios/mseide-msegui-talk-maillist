Hi,
Because of the new 'Home' button in file dialog some changes are necessary.
sys_gethomedir() has been renamed to sys_getapphomedir(). There is a new
sys_getuserhomedir() which returns the result of CSIDL_PROFILE query on
windows.
"~/" now returns sys_getuserhomedir() value, please use "^/" for
sys_getapphomedir().
Should sys_getapphomedir() return "<home>/.config/" if it exists on linux?
