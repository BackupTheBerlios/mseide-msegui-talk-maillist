does so only after some delay normally. It does _not_, though, if one
switches windows with the keyboard commands, normally.

about some time when the mouse cursor shifts to another window. It usually
settles on the selected window after, though it may _not_ be the upmost
window then (partially hidden beneath another one), and I even had the case
that _another_ window kept the focus.
