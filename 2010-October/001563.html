> What is these templates ? Their usage ?

Graeme uses them for example to insert skeletons into help definition files 
AFAIK. Every code template has a name, enter the name in the source editor, 
press Ctrl+T, the name will be replaced by the template text. Possible 
parameters can be separated by ',' in source editor or entered into a dialog.
