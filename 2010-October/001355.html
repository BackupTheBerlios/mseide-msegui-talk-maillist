>

MingW32 v6.8

>How do you start gdbserver? 
>

- put D:\mingw32\bin to the PATH variable & rebooted
- run the FAR manager & went to the MSEide.exe's directory
- called "gdbserver 192.168.1.2:9999 mseide.exe", on the WinXP
firewall pop-up request allowed gdbserver to listen,checked if it's
running (netstat -a)
