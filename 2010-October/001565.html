> What is these templates ? Their usage ?

Code Templates or Code Snippets. Ctrl+T select the template (*.mct files),
and the code is inserted into your source.

Here is a very simple TODO list code template. It inserts a code comment
which makes TODO items for me, it also correctly positions the edit cursor
for me. I have stacks more like this, and use them for fpdoc xml
documentation, creating classes or visitors for tiOPF, unit header
comments, a basic project template with a single form etc...


---------------[ todo.mct ]--------------------
[header]
name=todo
comment=Add a todo comment item
noselect=1
cursorrow=0
cursorcol=9
[]
{ TODO :  }
-----------------------------------------------


Save the above to <msegui>/apps/ide/templates/todo.mct

Restart the IDE and type:  todo
Then press Ctrl+T


The Code Template Editor dialog can be found by doing the following (I
don't know if it is available from somewhere else too).

  - Right click in your source editor
  - select 'insert template'
  - on the right edge of the grid click the '...' button
