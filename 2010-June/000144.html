> How do you use mailing lists, what is your setup?

Like normal emails. The only extra thing I did was to setup simple
filters in my email client to move mailing list messages out of my
default Inbox to various mailing list related folders (one folder for
each mailing list).  We have a IMAP server at work which I can access
from home too, so checking mail at work or home, means I don't have
duplicate messages, last read messages are correctly marked, message
bodies can be searched (unlike default NNTP setup where only subject
line can be searched) etc...
