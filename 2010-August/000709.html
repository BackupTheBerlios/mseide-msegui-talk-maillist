Graeme Geldenhuys wrote:

> Hi Martin,
> 
> I'm modified the source editor window so the active tab is a lot more
> visible. Is there a property I can modify in the tab component, so by
> default all tabs have this "more visible" colour defined?
> 
> More specifically, the Project Options have lots of tabs, and I find it
> hard to see the active tab, and would like to use the same thing as I have
> done for the Editor window. But I don't want to go and edit every single
> place in the IDE where tabs are used.
> 
> See attached screenshot for how my Editor Window tabs look.
> 
- Open apps/ide/guitemplates.pas.
- Click on 'skin' component.

The active tabs use cl_active by default. In order to change the RGB value
for cl_active 
- add an item to colors.count in objectinspector.
- set 'color' to "cl_active".
- set 'rgb' to the appropriate value.

In order to change the color of the tabs only
- change tabbar_*_tab_coloractive.
