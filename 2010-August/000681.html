Graeme Geldenhuys wrote:

> Hi Martin,
> 
> When I use the option 'Project > New > From Program', which template
> file does it use. I have setup a template project (with correct paths
> and search paths and macros) for use with fpGUI, but want the 'New >
> From Program' project to also have that paths and macros setup like
> the fpGUI project template.
> 
> So which project file do I need to overwrite?
> 
'Project'-'New'-'From Program' uses the settings of the current project, it
solely updates the project, mainprogram and exe names.
"
  else begin
   aname:= '';
   if filedialog(aname,[fdo_checkexist],'Select program file',
            ['Program files','All files'],
           ['"*.pas" "*.pp"','*'],'pas') = mr_ok then begin
    setcurrentdir(filedir(aname));
    with projectoptions do begin
     with t do begin
      mainfile:= filename(aname);
      aname:= removefileext(mainfile);
      targetfile:= aname+'${EXEEXT}'
     end;
     expandprojectmacros;
    end;
    aname:= aname + '.prj';
   end;
  end;
"
So first open your template and then 'Project'-'New'-'From Program' (not
tested). 'Project'-'New'-'From Template' should work too, MSEide does not
overwrite existing files (hopefully). ;-)
