It is not possible because params property is published and must be
persistent in order to allow design time settings and sometimes one wants
fix param indexes.
The params need not to be cleared before sending a new query AFAIK, missing
parameters are appended. One probably needs to use parambyname() because
the index depends on existing parameters.
