The property tfont.charset should bot be set for MSEgui applications because
MSEgui always uses unicode,tfont.charset is normally ignored. In order to
set fonts for MSEgui applications use
"
--fontalias=<alias>,<fontname>[,<fontheight>[,<fontwidth>[,<options>[,<xscale>]
                              [,<ancestor>]]]]
"
on the command line. <alias> = stf_default for the default base font.
