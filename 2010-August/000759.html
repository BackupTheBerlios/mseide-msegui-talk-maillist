The registration of *.mfm form resources is done by the accompanying
*_mfm.pas unit. MSEide rebuilds the *_mfm units if a *.mfm has been
modified. In order to set the modified flag without actually modifying the
form RightClick-'Touch Form' can be used. It is also possible to build the
*_mfm.pas files with tools/form2pas.
So the questions are:
- are the needed *_mfm units listed in uses of the form units?
- are the *_mfm units up to date?
