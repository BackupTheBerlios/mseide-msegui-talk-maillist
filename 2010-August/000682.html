Hi Martin,

When I do a 'Project > Build' I kept getting a "not decorated"
exception in one of my projects, but the project seems to have built
correctly. I then played with the Project Option "close window after
build". It was checked, so I unchecked it, so I can see if any errors
relating to this exception showed up in the Messages window. Then
suddenly the builds started working fine without any exceptions.
Checking the "close window after build" again, and still it's fixed. A
'self healing IDE' - cool!  :-)

No idea what caused it, but I thought I would mention the problem....
maybe you know the code that raises an "not decorated" exception.

I'm using r3836 and MSEide is compiled with FPC 2.4.3 (64-bit) running
on Ubuntu 10.04
