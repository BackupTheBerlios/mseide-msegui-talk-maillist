BTW:
Is there any existing feature request for code templates in MSEid; or does
it already exist - I just overlooked it?

I'm not looking for anything fancy like the latest Delphi IDE or Lazarus
IDE supports. Just something as simple as typing a few letters, press
Ctrl+J, and the IDE inserts a static code template and positions the cursor
where the template specified.

I use that a lot for fpdoc XML documentation, quickly inserting debug
lines, create visitor skeleton code for use with tiOPF, etc.

eg:
*  type "docel"  and press Ctrl+J, gives me a xml ELEMENT tag with short
   and long description nodes.

*  "docps" inserts the fpdoc PrintShort node

*  "docsa" inserts the fpdoc SeeAlso nodes

etc..
