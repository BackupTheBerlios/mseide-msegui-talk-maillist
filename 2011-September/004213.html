The FPC message output with -vu. If you want to produce the file in MSEide:
- 'Project'-'Options'-'Make'.
- Activate 'Copy messages to file'.
- Enter "test.txt" in 'Message output file'.
- 'Make options', add a -vu 'Command line options' entry.

Compile the project, in MSEunitdep select test.txt, click 'scan'.
