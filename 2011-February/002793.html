looks like you are trying to pass a component's property name rather than value.
<tmsesqlquery >. sql. add('select * from table1 where table1.id='
+tstringedit.value);

but you may need to quote the strings, so ...

const
qry_fmt = 'select * from "%s"="%s"'
var
qry: str;
...
qry :=Format(qry_fmt, ['table1.id',  tstringedit.value];

 <tmsesqlquery >. sql. add(qry);

is an option if you dont want to use parameters.
