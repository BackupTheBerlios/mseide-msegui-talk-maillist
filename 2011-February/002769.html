On Friday, 4. February 2011 16.06:50 Patrick Goupell wrote:
> I have a dbstringgrid that displays data from a mysql database.
>
> Can I add a new record in the dbstringgrid?
>
> What kind of options need to be set it it?  In the underlying query?  In
> the mysql database connection:?
>
Activate tdbstringgrid.optionsgrid og_rowinserting, og_autofirstrow and 
og_autoappend. Activate tmsesqlquery.controller.options dso_autoapply and 
dso_autocommitretaining.
