And what about under 64-bit Windows? Symlinks are not supported on all
Windows versions or all NTFS versions? Or does MSEide+MSEgui simply not
support 64-bit Windows at this point?

On a side note:
Personally, I think you got the back-end directory structure in MSEgui
all wrong. Your "back-end code" relates more to the "graphical API
backend", and not really to platform+OS code. That's why I choose 'x11'
and 'gdi' as my backend directories in fpGUI, instead of 'platform-os'.
Thus, this allows fpGUI to work on ARM Linux and WinCE devices too,
because 'gdi' and 'x11' still apply to those systems too.
