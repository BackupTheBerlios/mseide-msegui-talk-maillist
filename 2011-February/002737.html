On Tuesday, 1. February 2011 20.26:43 Sri Wahono wrote:
> How to play sound taudioout or tmidisource?
>
tmidisource dispatches midi events (note on, note off...), taudioout sends PCM 
data to the PulseAudio server. Both are used in the MSEgui signal processing 
framework (tab 'Math') which for instance can be used to build software sound 
synthesizers.
Midievents are available in tmidisource.ontrackevent. tsigmidisource can be 
connected with a serial midi port.
The PCM data must be delivered in taudioout.onsend as a dynamic array of 
samples.
