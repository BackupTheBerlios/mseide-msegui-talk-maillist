AFAIK, the usual TStringList way :

<tmsesqlquery>.active:= false;
<tmsesqlquery>.sql.clear; // only needed to replace previous SQL-commands
<tmsesqlquery>.sql.add('select * from table1');
[..]
<tmsesqlquery>.sql.add('select * from table1');
<tmsesqlquery>.active:= true;

AFAIK, for TMSESQLQuery, PrepareStatement option should be switched off if  
sevaral commands per list are put  ( because of DB servers are usually  
limited to single SQL-command per parameterized query)
